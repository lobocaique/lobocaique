# Portfolio Website

Personal portfolio built with React, TypeScript, and Vite.  
Showcases 2 ML & AI projects, tech stack, and experience.

## Live
https://lobocaique.github.io/lobocaique/

## Tech Stack
- React
- TypeScript
- Vite

## Content
- Project showcase with screenshots and descriptions  
- Responsive layout  

## Run locally

bash npm install npm run dev 

## Build

bash npm run build 

## Deployment

Deployed using GitHub Pages.

If deploying manually:
bash npm run build 

Then copy dist/ → docs/ and push.
